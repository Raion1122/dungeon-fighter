# 開発会議：攻撃エフェクト Phase 1 の実装範囲（2026-06-26）

**議題**: ユーザー作成 spec『攻撃エフェクト強化 仕様書 — Phase 1（DOM/CSS版）』を既存実装と照らし、実際に何を新規で作るかを決める。
**結論**: 候補①「重み2種に集中（踏み込み＋のけぞり）」＋ 被弾フラッシュの白化 を採択。

---

## 第1段：発散会議（候補出し）

### 第1ラウンド：ブレスト
- **【ガイウス】** 狙いは「打撃に重み」。だが数字も画面シェイクも既にある。欠けているのは“肉のぶつかり合い”＝攻撃側の踏み込みと被弾側ののけぞり。両者が棒立ちで数字だけ出ている現状を埋めるのが本丸。
- **【レンツ】** 爆弾。spec の5種、3種は既に動いている（`triggerEnemyDamageFlash`／`triggerScreenShake`=commit 2136e2f／`showDmgAt`）。spec 通り新規追加すると**ダメージ数字が二重に出る**。さらに `.df-recoil`/`.df-lunge` は CSS transform 依存だが、スプライトは**毎フレーム `el.style.transform="scaleX(-1)"`** で向きを上書き（player:3360／enemy:5679,5707／ally:8361）。クラスは1フレームで消える。14077 行に「transform は scaleX と競合するため不使用」と先人の注記まである。新規2種は per-frame 更新に offset を合成する方式（既存 `damageFlashTimer` 作法）でやる。
- **【ミサキ】** 踏み込み＋のけぞりは大賛成。加えて今の被弾フラッシュはセピアの退色ブリンク。spec が狙う「白くカッと光る」へ強化したい。通常ヒットにも軽い揺れが欲しい。
- **【ノエル】** 「通常ヒットにも揺れ」は待った。シェイクを crit/ボス/大呪文に絞ったのはオートバトルで毎ターン揺れると iPhone で酔うから。spec の `setTimeout(90)` は autoplay 高速巡回で遅延が積もる→`__autoplay` で必ずスキップ。フラッシュ白化は賛成、ただし新演出も既存 `screenShake` トグル＋`prefers-reduced-motion` に必ず乗せる。

### 第2ラウンド：絞り込み
- **【レンツ】** 流用＝flash/shake/数字。新規＝lunge/recoil（per-frame合成）。tavern.html は戦闘描画ゼロ＝**二重ファイル同期も不要**。index.html だけで閉じる。
- **【ノエル】** 二重化と全ヒット揺れを外せば地雷はほぼ消える。残る判断は「フラッシュ白化を今やるか」だけ。
- **【ガイウス】** 核（踏み込み＋のけぞり）は固定。フラッシュ強化はオプション。spec フル準拠は二重化承知の選択肢として残す。

### 候補一覧
- **①重み2種に集中（推し）**：lunge＋recoil のみ新規／flash・shake・数字は既存流用／per-frame transform 合成。小規模・低リスク・体感の伸び最大。
- **②フルパンチアップ**：①＋フラッシュ白化＋通常ヒットに微シェイク。中規模・中リスク（酔い／autoplay負荷）。
- **③仕様書フル準拠（dfPlayHit 単一エントリ）**：5種を spec 通り新規。Phase2/3 拡張性は最高だが数字二重化＋transform競合で衝突リスク大。
- **④演出トグル＆アクセシビリティ統合先行**：包括トグル整備→①。商用配布前の品質担保（地味）。

### 慶彦さんの決定
- スコープ＝**①**。フラッシュ＝**白化も足す（②寄り折衷）**。

## 第2段：収束会議（実装計画）

→ 詳細計画は plan ファイル `C:\Users\PC_User\.claude\plans\tender-squishing-hollerith.md` に確定。要点のみ:

- **per-frame transform 合成**：`composeSpriteTransform(flip, offPx)` で `scaleX(-1)` に lunge/recoil の translateX を合成。**左向き時は符号反転**（scaleX(-1) でX軸反転のため）。オフセットは `sin(p*π)`（p=1-T/Max）で行って戻り**厳密に0着地**。combatant 状態（`lungeT/Max/Dir`・`recoilT/Max/Dir`）として持ち per-frame 減衰。
- **流用**：`shakeEnabled()`（screenShakeトグル＋reduced-motion）を lunge/recoil のゲートに共有。`__autoplay` で完全スキップ。spec の `setTimeout(90)` は省略。
- **発火**：lunge＝`playerSingleAttack`/`allyBasicAttack` のスイング開始。recoil＝`triggerEnemyDamageFlash(index, dir?)` を拡張シグネチャ化（方向を渡したサイトのみ）＋ enemy→ally/player 被弾点。
- **フラッシュ白化**：`updateEnemyDamageFlash` のセピア filter→`brightness(2.6) drop-shadow white`。transform は不可侵（filter/opacity と責務分離）。色のみ＝非ゲート。
- **検証**：`driver_attackfx.js`（左符号反転・厳密0着地・各種スキップを決定論検証）＋ `driver_shake`/`driver_camera`/`driver_bgm` 回帰 ＋ iPhone 実機。
