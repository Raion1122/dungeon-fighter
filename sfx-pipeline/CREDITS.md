# 効果音(SFX) ライセンス方針

> 採用素材の **ID / ファイル / 出典 / ライセンス** の明細は、`build_sfx.py` が
> `assets/sfx/CREDITS.md` に自動生成する。本ファイルは**方針**を記す。

本ゲームは商用配布を想定するため、効果音は以下の方針で調達する。

## 採用してよいライセンス

- **CC0 / パブリックドメイン**: クレジット不要・商用可。第一候補 (Kenney, OpenGameArt の CC0, freesound の CC0 フィルタ)。
- **商用無料 (クレジット不要)**: 効果音ラボ 等。
- **CC-BY / 商用無料(クレジット必須)**: OtoLogic ("OtoLogic" 表記), 魔王魂 ("魔王魂" 表記) 等。
  → 使った場合は下記「クレジット必須素材」に追記し、**ゲーム内設定画面**(`audio.js` openSettings 末尾、
     VOICEVOX 表記の隣) にも表示すること。

## 取得方法と再配布

- `raw/` (元素材) は **コミットしない** (`.gitignore` 済)。リポジトリに入るのは整形後 mp3 のみ。
  CC0 でも元ファイルのまま大量同梱は避ける方針。
- 効果音ラボ / OtoLogic / 魔王魂 / くらげ工匠 への**スクレイピング・機械的一括DLはしない**
  (規約・サイト負荷への配慮)。必ず人間がブラウザでDLし `raw/inbox/<id>/` へ置く。
- freesound は **CC0 フィルタを掛けたもののみ** 採用。

## クレジット必須素材 (使用したら追記)

- (まだ無し — クレジット不要の CC0 / 効果音ラボ のみで揃える方針)
