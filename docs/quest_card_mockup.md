# 酒場のクエストカード — モックアップ仕様(2026-07-21 ユーザー提示)

ユーザーが ChatGPT に生成させた「羊皮紙の依頼札 6枚(3列×2行)」のモックアップを
文章で写し取ったもの。**画像ファイル自体は未保存**(必要なら `docs/quest_card_mockup.png`
としてユーザーに置いてもらう)。

## 絵柄の指定 ⚠️ プロンプトに必ず入れる

**ゲームブック『ファイティング・ファンタジー』シリーズの絵柄**。
= 1980年代 英国ゲームブックのペン画。純白黒のインク線画、密なクロスハッチング、
グロテスクで骨張った怪物、陰影は線の密度だけで表現、グレースケールの階調を使わない。

> 生成プロンプトでは**存命作家の個人名は書かず**、上記の画風記述で指定する
> (ChatGPT 側の拒否を避けるため、かつ記述の方が再現性が高い)。

モックアップ画像自体はセピア調に見えるが、これは**羊皮紙側の色**であり、
線画は黒インク。挿絵は透過 or 白背景の線画として生成し、CSS 側で
`mix-blend-mode: multiply` 等で羊皮紙に乗せるのが素直。

## 1枚のカードの構成(上から)

1. **リボンバナー** — 上端中央。左右に「クエスト」の小さな飾り文字、中央にドクロ紋章。
2. **タイトル** — 大きめ・中央寄せ(例: ゴブリンの討伐)。
3. **依頼人ブロック** — 左に依頼人のバスト肖像(ペン画)、右に `依頼人: 村長ブレン` +
   本文 3〜4 行。
4. 細い横罫線。
5. **`主な討伐モンスター`** — 小さいラベル(左寄せ)。
   下にモンスター名(大)+ 説明 2 行、**右側に大きめのモンスター挿絵**。
6. 細い横罫線。
7. **`報酬`** — ラベル + 銭袋アイコン + `120 ゴールド` + XPバッジ + `200 経験値`。
8. **右下に小さなアイテムアイコン**(カードごとに異なる: 本 / ドクロ / 指輪 など)。

紙質: 羊皮紙、四隅と縁に焼け・シミ・折れ。インクは焦茶〜黒。

## ⚠️ 実装上の含意

- **カードを丸ごと 1 枚絵で生成するのは破綻する。** `QuestGen` がクエストを組合せで
  無限生成するため、文字を画像に焼き込めない。
  → 生成するのは **部品**(羊皮紙テクスチャ / 依頼人肖像 / モンスター挿絵 /
  銭袋・XP・アイテムのアイコン / リボンバナー)で、**文字と組版は HTML+CSS**。
- モックの 6 件は既存の固定 6 シナリオとは別物(生成クエスト側の例)。
  固定シナリオには伏線・隠し要素が紐づくので**併存設計**が要る。
